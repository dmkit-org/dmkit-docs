# Battle Map

The battle map is a grid-based canvas for running encounters. Access it by double-clicking an Area card.

## The Grid

The map shows a square grid. The default cell size is 50px and can be adjusted in **Map Settings**. Tokens snap to the grid automatically.

**Navigating the map:**
- **Pan** — middle mouse button or two-finger drag
- **Zoom** — scroll wheel or pinch

## Placing Tokens

The **Entity Panel** on the right side of the battle map shows your campaign's entity library.

1. Search or filter by type (Character, Item, Hazard)
2. Drag an entity from the panel onto the map
3. A token appears linked to that entity, snapped to the nearest grid cell

You can place multiple tokens from the same entity — useful for groups of identical enemies.

## Moving Tokens

Drag any token to move it to a new grid cell. Position is saved automatically.

## Token Detail

Click a token to open its detail panel:

- View the linked entity's stats (HP, AC, ability scores)
- Edit **current HP** — cannot exceed the entity's maximum
- Add or remove **conditions** (poisoned, blinded, stunned, etc.)

Changes to a token's state do not affect the base entity in the library.

## Deleting Tokens

Right-click a token and select **Remove** to delete it from the map. This does not delete the entity from the library.
