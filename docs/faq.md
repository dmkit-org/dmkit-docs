# FAQ

## Account

**Can I sign in with Google?**
Yes. On the sign-in screen, click **Sign in with Google** to authenticate with your Google account.

**Can I change my email or password?**
Email and password changes are not yet supported from within the app. If you need help, contact support.

## Campaigns

**Are my campaigns private?**
Yes. Campaigns are private to your account. No other user can see or access them.

**What happens when I delete a campaign?**
Deleting a campaign permanently removes all its data — Worlds, Locations, Areas, Entities, Factions, Quests, Session Notes, and Player Characters. This cannot be undone.

**Can I back up my campaigns?**
Yes. Use **Export Campaign** from the campaign menu to download a full JSON backup. You can re-import it at any time to restore it as a new campaign.

## Battle Map

**Can I place the same entity multiple times on a map?**
Yes. You can drag the same entity from the Entity Panel onto the map multiple times — useful for groups of identical enemies.

**Do changes to a token affect the entity in my library?**
No. Editing a token's HP or conditions only affects that token instance on the map. The base entity in the library is unchanged.

## Export & Import

**What does the export file contain?**
Everything in the campaign — Worlds, Locations, Areas, token instances, all entity library entries, Factions, Quests, Session Notes, and Player Characters.

**Can I share an export with someone else?**
Yes. Another DMKit user can import your export file to create their own independent copy of the campaign.
